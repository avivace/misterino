# Mister Notifier

Telegram bot notifying when your favorite streamers go live on Twitch.tv.


## Get started

```bash
# System dependencies
apt install python3 python3-pip python3-venv

# Get things ready
git clone git@github.com:avivace/mister-notifier.git
cd mister-notifier
python3 -m venv .
source bin/activate
pip3 install -r requirements.txt

# Start the Bot
python3 bot.py
```
